# 常见问题与修复

## "字幕滞后于音频"（最常见）

**症状**：字幕比音频晚4-15秒出现。

**根因**：场景边界设得太晚，或字幕间距太宽。

**修复**：
1. 重新从 timing_reference.md 锚点表计算场景边界
2. 压缩场景内字幕间距
3. 将受影响场景的所有字幕整体前移

## "字幕停留时间不对"

**症状**：长文本字幕在音频还在说话时就消失了；短字幕停留过久。

**修复**：应用文本长度公式（2-5字→2s, 6-12字→2-2.8s, 13-20字→2.8-3.5s, 20+字→3.5-4s）。绝不用统一时长。

## "BGM没有声音"

**症状**：渲染视频只有TTS，无背景音乐。

**修复**：
1. `ls -la composition/bgm/` 确认是实体文件非 symlink
2. 提高 `data-volume` 从 0.15 到 0.30-0.35
3. 检查渲染输出中 BGM 是否有 404 错误
4. 检查同 track index 的 BGM 没有时间重叠

## "场景不切换"

**症状**：整个视频只显示第一个场景。

**根因**：GSAP timeline 未正确注册，或场景 opacity 管理出错。

**修复**：
1. 确认 `window.__timelines["id"] = tl` 的 id 与 root div `data-composition-id` 一致
2. 确认场景2+ 在 style 属性中有 `opacity: 0`
3. 确认所有 `tl.to()` 和 `tl.fromTo()` 引用了正确的场景 ID

## "静音检测给出错误边界"

**症状**：静音检测得出的场景边界与实际语音不匹配。

**根因**：静音检测把句内停顿（0.5-0.8s）误判为场景边界。

**修复**：用 timing_reference.md 锚点表作为场景时间唯一来源。静音检测仅作参考。

## "字幕文字与音频不匹配"

**症状**：字幕显示的文字和TTS实际说的不一致。

**修复**：将所有字幕文本与 `tts_script.md` 逐条 diff。移除不在TTS脚本中的字幕行，补充TTS脚本中有但字幕遗漏的行。

## "字幕整体偏移"

**症状**：视频后半部分字幕全部延迟2-4秒。

**根因**：某个句子的起始锚点算错，导致后续所有字幕累积偏移。

**修复**：回到 timing_reference.md 逐句时长表，核对每句的累积起始时间，修正错误的锚点。
